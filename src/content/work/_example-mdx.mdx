## MDX features

Because this file is `.mdx` rather than `.md`, you can import and use Astro components inline.

For example, you could embed an interactive dockets table, a citation graph, or a press-mentions list rendered from the `press` collection.

### Using backlinks

Reference other works in the corpus with the wikilink syntax `[[other-work-slug]]`. At build time, the work detail page for that slug will automatically show this work as a backlink.

For example, the [[rewiring-utility-planning]] report and the [[gpsc-georgia-power-2022-irp]] testimony will both list this page as a backlink.
